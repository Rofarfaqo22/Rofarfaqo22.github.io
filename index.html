Github & Git

Langkah pembuatan github page dengan Git

Langkah 1 : Git Init

Melakukan inisiasi folder dengan ‘git init’ sehingga folder yang ingin kita push nanti otomatis menjadi repo-git.

Langkah 2 : Git Status

Mengecek status files yang ada pada folder tersebut dengan git status

Langkah 3 : Git Add

Melakukan git add . untuk menambahkan file ke staging area

Langkah 4 : Git Commit

Melakukan commit pertama untuk menyimpan perubahan yang terjadi pada folder kita

Langkah 5 : Git Remote

Melakukan git remote repository yang sudah kita siapkan untuk menghubungkan repo git dengan repo local

Langkah 6 : Git Push

Melakukan push file yang sudah diadd tadi ke repository yang sudah disiapkan. Sehingga file akan ditambahakan ke repository github
